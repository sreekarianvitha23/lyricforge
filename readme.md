# 🎵 LyricForge — AI Lyrics Generator

> Turn your feelings into song — instantly

LyricForge is an AI-powered lyrics generator that helps anyone create original song lyrics in seconds. Just pick a mood, genre, and theme — and get creative lyrics tailored to you.

---

## ✨ Features

- 🎭 **8 Moods** — Hopeful, Romantic, Heartbroken, Empowering, Nostalgic, Angry, Dark, Chill
- 🎸 **7 Genres** — Pop, R&B, Hip-Hop, Rock, Country, Folk, EDM
- 📝 **4 Song Structures** — Full Song, Short, Verse Only, Chorus Only
- 🎤 **Artist Style** — Customize lyrics to match your favorite artist's vibe
- 📋 **One-click Copy** — Copy your lyrics instantly to clipboard
- ⚡ **No backend needed** — Runs entirely in the browser

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/sreekarianvitha23/lyricforge.git
```

### 2. Open the project folder
```bash
cd lyricforge
```

### 3. Run it
Just open `index.html` in your browser — no installation needed!

## 🎯 How It Works

1. Enter a **theme or story** for your song
2. Choose a **genre** and **song structure**
3. Select your **mood** from the chip selector
4. Optionally add an **artist style**
5. Hit **Generate Lyrics** ✦
6. Copy and use your lyrics!
